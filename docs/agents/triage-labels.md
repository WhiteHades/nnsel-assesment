# Triage labels

Use the default label vocabulary.

## Labels

- `needs-triage`: maintainer needs to evaluate
- `needs-info`: waiting for missing context
- `ready-for-agent`: ready for agent implementation
- `ready-for-human`: needs human implementation or review
- `wontfix`: will not be actioned

## Use

Apply only the labels that match the current state. Do not create duplicate labels with different names unless the user asks for a new vocabulary.
